What you're watching

The input is the least impulse-like thing there is. Every one of the 2ⁿ amplitudes has the same magnitude; all your information lives in the phase θ(x) = Σⱼ φⱼ·xⱼ. Because that phase is additive over the bits, n knobs only reach an n-parameter family of signals — you steer where the energy lands and how sharp it is, not an arbitrary shape.

The smooth curve is the ideal state; the bars sample it. An infinite-resolution machine would hold the continuous spectrum |V(κ)|². A real n-qubit register only has 2ⁿ slots, so it reads off that curve at the integer bins. When the ramp lands the peak exactly on an integer, every other bin sits on a zero of the curve → a single clean spike. Land it between integers and the bins sample the skirts → spectral leakage.

The leak law. Detune one qubit's phase by δ and the peak drops to exactly cos²(δ/2), the rest bleeding into neighbours — same for any qubit, because each splits the indices evenly.

The axis wraps around. Frequency bin N is the same state as bin 0 — the spectrum is periodic. So when the peak sits at |0…0⟩, the curve also climbs at the far-right edge: that's the same spike seen through the wrap, not a second peak at |1…1⟩. Trust the orange dots, not the rising skirt.

The math QFT, formally
Input state — flat magnitude, phase set by your n rotations
$$|\psi_{\text{in}}\rangle=\frac{1}{\sqrt{N}}\sum_{x=0}^{N-1} e^{\,i\theta(x)}\,|x\rangle,\qquad \theta(x)=\sum_{j=0}^{n-1}\varphi_j\,x_j,\quad N=2^{\,n}$$
QFT amplitude on basis state $|k\rangle$
$$V[k]=\langle k|\,\mathrm{QFT}\,|\psi_{\text{in}}\rangle=\frac{1}{N}\sum_{x=0}^{N-1} e^{\,i\theta(x)}\,e^{-2\pi i\,kx/N}$$
Measured probability and its smooth continuation $|V(\kappa)|^2$
$$P(k)=\bigl|V[k]\bigr|^2,\qquad \kappa\in[0,N)\ \text{continuous}$$
Clean-tone condition — a 1:2:4:8 ramp is a single frequency
$$\varphi_j=\alpha\,2^{\,j}\;\Rightarrow\;\theta(x)=\alpha x\;\Rightarrow\;\text{peak at }k=\frac{\alpha N}{2\pi}$$
Leak law — cost of detuning qubit $j$ by $\delta_j$
$$P_{\text{peak}}=\prod_{j=0}^{n-1}\cos^2\!\Bigl(\tfrac{\delta_j}{2}\Bigr)$$
Periodicity — why the curve rises again on the right
$$V[k+N]=V[k]\quad\Longleftrightarrow\quad |k\rangle\ \text{and}\ |k+N\rangle\ \text{label the same bin}$$
